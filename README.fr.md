# CiteLibre Particip'EZ
Ce repository héberge tout le code source requis pour lancer une application CiteLibre ParticipEZ locale complète.

CiteLibre vous permet de mettre en place facilement une politique de participation citoyenne avec des services de Déclaration d'incidents sur le domaine public, des collicitations spontanées, des appels à projets avec publications des idées pouvant être éligibles aux votes...
